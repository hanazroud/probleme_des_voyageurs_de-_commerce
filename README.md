# Algorithme Évolutionnaire – Problème du Voyageur de Commerce avec Tabou 

Ce projet implémente un algorithme évolutionnaire en Python pour résoudre le **problème du voyageur de commerce (TSP)**.  
Il affiche le **meilleur chemin trouvé** entre les villes et la **distance minimale correspondante**.
